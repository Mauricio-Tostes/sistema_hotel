# Sistema Hotel

Sistema de cadastro e gestão de clientes para hotel, desenvolvido em **Python** com interface gráfica em **Tkinter** e banco de dados **SQLite**.

## Objetivo

Projeto inicialmente inspirado e guiado pelo curso do desenvolvedor **Rafael Serafim** no YouTube.  
A partir dessa base, o sistema foi expandido com diversas melhorias visuais e funcionais, servindo como prática de programação e estudo de boas práticas em desenvolvimento.

## Tecnologias utilizadas

- Python 3
- SQLite (banco de dados local)
- Tkinter (interface gráfica)
- ReportLab (geração de relatórios em PDF)
- Requests (consulta de CEP via API)

## Funcionalidades atuais

- Cadastro, alteração e exclusão de clientes
- Validação de CPF com algoritmo oficial
- Formatação automática de telefone e data de nascimento
- Consulta de endereço via CEP (API ViaCEP)
- Busca de clientes por nome ou CPF
- Geração de relatórios em PDF
- Campos com validação e formatação automática
- Interface gráfica aprimorada para melhor usabilidade

## Créditos

Projeto desenvolvido inicialmente com base no curso do desenvolvedor **Rafael Serafim** disponível no YouTube, e posteriormente aprimorado com novas funcionalidades e melhorias próprias.
