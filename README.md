# Sistema Hotel

Sistema simples de cadastro para hotel utilizando Python e SQLite.

## Objetivo

Projeto desenvolvido para fins de estudo e prática de programação em Python.

## Tecnologias utilizadas

- Python
- SQLite
- Tkinter (interface gráfica)

## Funcionalidades Futuras

- Integração com API para preenchimento automático de endereço a partir do CEP.

## Créditos

Projeto desenvolvido com base no curso do desenvolvedor Rafael Serafim disponível no YouTube.
